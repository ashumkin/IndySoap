TIdSoapInteger

unit: IdSoapTypeRegistry

The TIdSoapInteger type declares aTIdBaseSoapableClass descendent that exposes an integer. This class exists to allow soap consumers to be informed of the difference between an integer of value 0, and an integer where the xml atttribute nil="true" is defined.

The WSDL importer will use this type where the WSDL declares that a type is nillable. You can interchange between integer and TIdSoapInteger in the interface declaration.
