TIdSoapString

unit: IdSoapTypeRegistry

The TIdSoapString type declares aTIdBaseSoapableClass descendent that exposes a string. This class exists to allow soap consumers to be informed of the difference between an empty string, and a string where the xml atttribute nil="true" is defined.

The WSDL importer will use this type where the WSDL declares that a type is nillable. You can interchange between string and TIdSoapString in the interface declaration.
