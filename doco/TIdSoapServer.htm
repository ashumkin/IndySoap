TIdSoapServer

Ancestor: TIdSoapListener

Unit: IdSoapServer

TIdSoapServer is the base class for all classes that provide SOAP request/response services.

For properties and functional documentation, refer to TIdSoapListener

 

  procedure HandleSoapRequest(AInMimeType: String; ARequest, AResponse: TStream; var VOutMimeType : string): Boolean;

This function is the primary Soap entry point for transport layers.
Parameters:

AInMimeType The mime type sent by the original requesting system. If this is blank then the Server will try to guess at the encoding
ARequest The Original request from the Client
AResponse An empty stream. the server will place it's response here to be returned to the client. The server will reset the position of this stream to 0 when it is finished
VOutMimeType The mime type to send back to the client with the response. This is a guess based on the MimeType supplied and the encoding used in the response stream
result false if the soap packet being returned in the Response Stream contains a Soap Fault. True otherwise. The transport layer may need to know this for it's purposes (i.e. HTTP returns 500 in this case)
