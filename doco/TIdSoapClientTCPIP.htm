TIdSoapClientTCPIP

Ancestor: TIdSoapBaseClient

Unit: IdSoapClientHTTP

Provides a SOAP mapping to a custom IndySoap direct TCP/IP communication protocol for use with SOAP request/response. The major advantage of this protocol is that a connection is held open, and the protocol is extremely light weight. This makes this protocol suitable for use in a LAN environment with tightly bound client/server environment.

The connection is opened and closed as the component active property is changed

 

  property SoapHost : string;

Network address or name of SOAP Server

 

  property SoapPort : word;

Listening Port of SOAP Service

 

  property SoapTimeout : integer;

How long to wait before giving up on the server

 

 property TCPClient : TIdTCPClient;

Summary: Ignore SOAPHost and SOAPPort, and use the assigned TCPClient instead.
