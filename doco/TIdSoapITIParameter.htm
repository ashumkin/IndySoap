TIdSoapITIParameter

TIdSoapITIParameter is part of the ITI structure, storing information concerning a particular Parameter of a method. The ITI stores all the metadata associated with the interfaces in use by IndySoap.

You should not use a TIdSoapITIParameter directly. You may see an IndySoap leak notification concerning TIdSoapITIParameter - if this is so then either a client or server has not been cleaned up
