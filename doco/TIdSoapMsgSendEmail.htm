TIdSoapSendEmail

Ancestor: TIdSoapBaseSender

Unit: IdSoapMsgEmail

TIdSoapClientDirect provides a One way client implementation that sends the SOAP Messages as EMail to an SMTP server.

 

  property Destination : TIdEMailAddressItem;

the email address to send the messages too. You must provide an address. The other details are optional

 

  property SMTPHost : String; 
  property SMTPPort : word;

The details of the SMTP Server to use

 

  property SMTP : TIdSMTP;

If you need to set advanced settings on the SMTP Client object, create another one, and point this property at it. This alternative SMTP object will be used instead
