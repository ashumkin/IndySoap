TIdSoapWebBroker

Ancestor: TIdSoapServer

Unit: IdSoapWebBroker

TIdSoapWebBroker implements a Web Broker component for IndySoap, suitable for use on WebModules.

To use the IndySoap WebBroker Server, drop it on a WebModule. You need to set 2 sets of properties:

  • Standard IndySoap properties. Consult TIdSoapServer for further information.
  • WebDispatch Path. You wouldn't normally need to set this as the default value soap* is usually appropriate

Generally, you also need a TIdSoapWebBrokerWSDL to publish the WSDL for the service

 

You can have multiple different Servers on different paths, each publishing a different set of SOAP services.
