TIdSoapClientDirect

Ancestor: TIdSoapBaseClient

Unit: IdSoapClientDirect

TIdSoapClientDirect provides a Request/Response client implementation that connects directly to the SOAP Server. This is useful for DUnit testing, debugging, and in process deployment (though you should consider using the interfaces directly if you are in process).
