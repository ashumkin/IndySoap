TIdSoapBaseClient

Ancestor: TIdSoapBaseSender

Unit: IdSoapClient

 

TIdSoapBaseClient is the base object for the client side implementation of the Soap V1.1 standard for RPC request/Responses. You do not use TIdSoapBaseClient directly. Instead, you use a descendent that also provides a transport implementation.

TIdSoapBaseClient introduces no new properties or behaviour. See TIdSoapSender for more information
