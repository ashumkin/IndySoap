TIdSoapMsgSMTPListener

 

TIdSoapMsgSMTPListener hosts a SOAP Server that processes incoming SMTP EMail messages as SOAP messages. SOAP messages are processed in the SMTP Server thread.

There is no IdCustomSMTPServer, so the TIdSoapMsgSMTPListener object descends straight off the the TIdSMTPServer. This event publishes all the normal events of a TIdSMTPServer, but you cannot use any of the following properties / events:

  • OnReceiveRaw
  • OnReceiveMessage
  • OnReceiveMessageParsed
  • ReceiveMode

using any of these will render the server inoperable. The only SOAP Specific property that you have to set is

 

  property Listener: TIdSoapMsgReceiver;

The SOAP Listener that provides SOAP messaging services
