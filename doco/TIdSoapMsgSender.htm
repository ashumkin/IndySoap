TIdSoapBaseMsgSender

Ancestor: TIdSoapBaseSender

Unit: IdSoapClient

 

TIdSoapBaseMsgSender is the base object for the client side implementation of the Soap V1.1 standard for RPC one way messaging. You do not use TIdSoapBaseMsgSender directly. Instead, you use a descendent that also provides a transport implementation.

TIdSoapBaseMsgSender introduces no new properties or behaviour. See TIdSoapSender for more information
