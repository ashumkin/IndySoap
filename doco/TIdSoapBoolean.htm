TIdSoapBoolean

unit: IdSoapTypeRegistry

The TIdSoapBoolean type declares aTIdBaseSoapableClass descendent that exposes a boolean. This class exists to allow soap consumers to be informed of the difference between a default boolean (false), and a boolean where the xml atttribute nil="true" is defined.

The WSDL importer will use this type where the WSDL declares that a type is nillable. You can interchange between boolean and TIdSoapBoolean in the interface declaration.
