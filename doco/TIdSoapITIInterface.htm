TIdSoapITIInterface

TIdSoapITIInterface is part of the ITI structure, storing information concerning a particular interface. The ITI stores all the metadata associated with the interfaces in use by IndySoap.

You should not use a TIdSoapITIInterface directly. You may see an IndySoap leak notification concerning TIdSoapITIInterface - if this is so then either a client or server has not been cleaned up
