TIdSoapQName

unit: IdSoapTypeRegistry

The TIdSoapQName is an implementation of the XML W3C schema type QName. This has 2 parts, a name and a namespace. where the namespace is integrated in the XML namespace system.

You would need to use this where you require a QName value in the message. All you need to do is populate or read the properties. IndySoap will handle the namespace integration
