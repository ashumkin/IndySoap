TIdSoapMsgReceiver

Ancestor: TIdSoapListener

Unit: IdSoapServer

TIdSoapMsgReceiver is the base class for all classes that receive and process incoming SOAP messages without sending a response.

For all properties and functional documentation, refer to TIdSoapListener

 

  procedure HandleSoapRequest(AInMimeType: String; ARequest: TStream): Boolean;

This function is the primary Soap entry point for transport layers.
Parameters:

AInMimeType The mime type sent by the original requesting system. If this is blank then the Server will try to guess at the encoding
ARequest The Original request from the Client
AResponse An empty stream. the server will place it's response here to be returned to the client. The server will reset the position of this stream to 0 when it is finished
