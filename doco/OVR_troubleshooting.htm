Troubleshooting

So many things can wrong with Soap that it can be difficult to know where to start.

 

Pascal -> ITI conversion

The pascal -> ITI conversion process is pretty reliable. Generally, if you get an error in this process, you've attempted to do something that indysoap cannot support.

 

WSDL -> Pascal conversion

If the problem concerns the WSDL to Pascal conversion process failing, there's almost certainly something in the WSDL that IndySoap does not support. You can forward the (whole) WSDL to the IndySoap team in a bug report for consideration. But there's many things that WSDL's declare that cannot be supported through IndySoap, so it may not be accepted as a bug

 

Run time SOAP problems

 

In general, if you want to submit a bug report to the IndySoap team, you will need to assemble some resources. Often, the process of assembling these resources will help you solve the problem. But there's several things to check before you do:

  • Is the ITI current?
  • Are all your implementation methods published?
  • Are all the types registered?
  • Does your implementation object descend from TIdBaseSoapImplementation
  • If the interface source wasn't generated by the IndySoap WSDL -> Pascal convertor, check the interface source carefully. Also check that the appropriate SOAP Metadata has been added to the interface declaration
  • check the properties of the soap components.

If you want to report a bug to IndySoap, you generally need to submit the following things:

  • the WSDL for the server
  • the interface source
  • a packet trace for both request and response
    The easiest way to generate a packet trace is to use proxyTrace or tcpTrace. Both are available from http://www.pocketsoap.com
  • a detailed description of the problem

Report all bugs to the IndySoap mailing List. If there is some reason why you are not happy to attach these resources to the bug report, you can indicate that you are prepared to supply them on request to an IndySoap team member when they are working on your bug.
