TIdSoapITIMethod

TIdSoapITIMethod is part of the ITI structure, storing information concerning a particular method. The ITI stores all the metadata associated with the interfaces in use by IndySoap.

You should not use a TIdSoapITIMethod directly. You may see an IndySoap leak notification concerning TIdSoapITIMethod - if this is so then either a client or server has not been cleaned up
