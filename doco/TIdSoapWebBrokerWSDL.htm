TIdSoapWebBrokerWSDL

Ancestor: TIdSoapComponent

Unit: IdSoapWebBroker

TIdSoapWebBrokerWSDL implements a Web Broker component that publishes WSDL's for IndySoap, suitable for use on WebModules.

To use the IndySoap WebBrokerWSDL publisher, drop it on a WebModule. You need to set 2 properties:

  • Server. This should be a TIdSoapWebBroker on the same webmodule. The WSDL publisher will publish the services provided by this server
  • WebDispatch Path. You wouldn't normally need to set this as the default value wsdl* is usually appropriate
