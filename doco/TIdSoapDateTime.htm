TIdSoapDateTime

unit: IdSoapDateTime

TIdSoapDateTime encapsulates an XML schema datetime. XML date times differ from TDateTime in 2 fundamental respects.

The first is that some portions of the date precision may be empty. TDateTime cannot represent this.

The second is that the XML schema datetime type includes timezone information
